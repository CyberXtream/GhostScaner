About GhostScaner

GhostScaner is a powerful, multi-threaded network discovery tool built in Python that allows you to scan your local network and identify all connected devices. This tool provides detailed information including IP addresses, MAC addresses, hostnames, and connection status of devices on your network.

Prerequisites

  • Python 3.6+ - Make sure Python is installed on your system
  • Scapy Library - For network packet manipulation
  • Administrator Privileges - Required for low-level network access

Installation

1. Install Python Dependencies

pip install scapy

2. Windows-Specific Setup (Recommended)

For optimal performance on Windows, install Npcap:

  1. Download Npcap from npcap.com
  2. During installation, select "Install Npcap in WinPcap API-compatible mode"
  3. Restart your computer after installation

Usage

Running the Scanner

python network_scanner.py

Example Workflow

  1. Run the script with administrator privileges
  2. Enter your network in CIDR notation (e.g., 192.168.1.0/24)
  3. Choose whether to save results to a file
  4. Review the discovered devices

Troubleshooting

WinPcap/Npcap Error

Install Npcap as described above. The scanner will use fallback methods if unavailable.

No Devices Found

Verify network connection, correct CIDR notation, and try running with administrator privileges.

Permission Errors

Run Command Prompt/PowerShell as Administrator on Windows, or use sudo on Linux/macOS.
